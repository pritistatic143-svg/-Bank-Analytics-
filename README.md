# Bank Analytics Dashboard

## Project Overview
This project analyzes banking and loan customer data to identify loan trends, repayment behavior, customer segmentation, and financial insights using Excel, SQL, Power BI, and Tableau.

## Tools Used
- Excel
- SQL
- Power BI
- Tableau

## KPIs
- Year-wise Loan Amount Statistics
- Grade and Sub-Grade wise Revolving Balance (revol_bal)
- Total Payment for Verified Status vs Non-Verified Status
- State-wise and Month-wise Loan Status Analysis
- Home Ownership vs Last Payment Date Statistics
- Total Loan Amount
- Total Payments Received
- Loan Status Percentage Analysis
- Purpose-wise Loan Distribution
- Fully Paid vs Charged Off Loan Analysis

## Insights
- Identified yearly growth trends in loan amounts
- Analyzed revolving balance across grades and sub-grades
- Compared payment behavior between verified and non-verified customers
- Evaluated loan status trends across different states and months
- Studied impact of home ownership on payment behavior
- Monitored loan repayment and customer risk patterns

## Dashboard Features
- Interactive Filters and Slicers
- Year-wise Trend Analysis
- State-wise Loan Monitoring
- Grade-wise Customer Analysis
- Loan Status Tracking
- Customer Segmentation Dashboard
